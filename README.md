🧑‍💻 Atividade 1 – Cartão de Perfil Pessoal
Objetivo
Criar uma página simples apresentando um perfil pessoal.

Requisitos HTML
A página deve conter:

Um título principal com o nome da pessoa.
Uma foto (pode ser uma imagem da internet).
Um parágrafo de apresentação.
Uma lista com: Hobby favorito, comida favorita e filme/série favorita
Requisitos CSS

Aplicar:
Cor de fundo da página.
Cor do texto.
Fonte personalizada.
Borda na imagem.
Margem nos elementos.
Padding em pelo menos um elemento.
Título centralizado.
Desafio Extra
Criar uma caixa de destaque para a apresentação utilizando:
border
background-color
padding
Instruções de Entrega:

Criar um repositório no GitHub.
Subir o conteúdo desenvolvido para o repositório. Criar um documento de texto com o link do repositório público. Enviar o documento de texto contendo o link do repositório aqui na atividade do Classroom. Cada membro do grupo deve fazer sua própria entrega. Caso o contrário, não será avaliado.
